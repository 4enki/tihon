# Morris

## Режимы
Запуск живой сборки на локальном сервере с вотчерами:

``` bash
npm run start
```

Одноразовая сборка:

``` bash
npm run one
```

Сборка проекта и архивация в _*.zip_-архив:

``` bash
npm run zip
```
